# Lamborghini Sales Data Analysis

## Description
Exploratory data analysis project focused on car sales data.
Includes data cleaning, aggregation, SQL integration, and visualization.

## Tech Stack
- Python
- Pandas
- SQLite
- Matplotlib

## Features
- Data loading and cleaning
- SQL database integration
- Analytical queries
- Aggregation and grouping
- Visual analysis

## Purpose
Demonstration of data analytics skills and analytical thinking.
